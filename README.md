# Hyprland Dotfiles

Hyprland Rice. Created for personal use in Cachyos.

**Disclaimer: This is for personal use and was made on an MSI laptop.**

## 📦Programs

List of programs used

| Component         | Program        |
| ----------------- | -------------- |
| Window Manager    | hyprland       |
| Terminal          | kitty          |
| Shell             | zsh            |
| Fetch             | fastfetch      |
| File Manager      | nautilus (TBD) |
| Browser           | zen browser    |
| Bar               | waybar         |
| Launcher          | vicinae        |
| Lockscreen        | hyprlock       |
| Login Menu        | sddm           |
| Notification Menu | swaync         |
| Powermenu         | rofi           |

## Extras

Install these extensions for vicinae:

- AWWW Switcher
- Bluetooth
- Arch Packages
- Player Pilot
- hypr keybinds
- Power profile
- Hyprland Monitors
- Spotify Player
- Wifi Commander
- PulseAudio
- NerdFonts
